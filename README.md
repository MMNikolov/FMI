# FMI Coursework & Programming Exercises

A centralized repository containing university coursework, lecture materials, and programming assignments completed during my Informatics degree at the Faculty of Mathematics and Informatics (FMI).

---

## 📂 Repository Structure

*   **`OOP/`** – Object-Oriented Programming code, assignments, and design patterns. Contains core logic for state management and component communication.

---

## 🛠️ Key Implementations

The **`OOP`** module contains custom programming architectures focusing on clean design and hardware/state emulation:

*   **Control Hubs:** Centralized management systems to route data and handle environment inputs.
*   **State & Component Control:** Logic for smart environment utilities, including thermostats, static visual displays, and system toggles.
*   **Object-Oriented Design:** Practical application of encapsulation, inheritance, polymorphism, and interface abstraction.
